User Crud

This project was completed as part of the coding challenge for the Code Fellows Python Development Accelerator in order to show a basic familiarity with Django development.

Deployed on Heroku, the app allows visitors to create, read, update, and destroy users stored in a Postgres database. Additionally, select users can be protected from casual deletions or modifications through the admin panel.

There also may be a few easter eggs included--just for fun.
